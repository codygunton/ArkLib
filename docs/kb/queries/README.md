# Query Pages

Use this directory for answers that started as a question or review task but are worth preserving.

Typical examples:

- "Which theorem in paper X corresponds to which Lean declarations?"
- "How do STIR and WHIR differ in the current ArkLib formalization?"
- "Which reference version should reviewers use for this PR?"

If the result is durable and likely to be reused, file it here instead of leaving it in chat.
