# JM24

## At A Glance

`JM24` is one of the two current references named in ArkLib's Algebraic Group Model development.
It provides modern AGM/GGM transfer context for the repository's attempt to mechanize algebraic
group-model adversaries and oracle interfaces.

## What ArkLib Uses From This Paper

- Background and modeling guidance for the AGM setup in
  [`ArkLib/AGM/Basic.lean`](../../../ArkLib/AGM/Basic.lean).
- Context for how algebraic-group-model proofs may relate to more generic computation models.

## Main ArkLib Touchpoints

- [`ArkLib/AGM/Basic.lean`](../../../ArkLib/AGM/Basic.lean) cites `JM24` directly in its module
  docstring alongside `LPS24`.

## Version Notes

- ArkLib currently uses `JM24` as a modeling reference, not as a line-by-line theorem target.
- This page should be read together with [`LPS24.md`](LPS24.md) for the current AGM subtree.

## Known Divergences From ArkLib

- The current ArkLib AGM file is still a foundational API file with incomplete proofs and TODOs.
- The repository focuses on explicit oracle interfaces and adversary execution rather than on the
  full theorem suite of the paper.

## Open Formalization Gaps

- `ArkLib/AGM/Basic.lean` still contains incomplete definitions and proofs, so this reference
  currently anchors design intent more than finished formal theorems.
