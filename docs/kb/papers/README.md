# Paper Pages

This directory contains canonical markdown pages for BibTeX keys that are cited from Lean files or
are otherwise active in ArkLib development.

The naming convention is:

- `KEY.md`

Examples:

- `BCIKS20.md`
- `ACFY24.md`
- `DP24.md`

Each page should explain:

- what the paper is;
- which parts ArkLib uses;
- which modules currently depend on it or cite it;
- where ArkLib intentionally differs from the paper;
- what formalization gaps remain.
