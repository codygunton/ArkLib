# PS94

## At A Glance

`PS94` is one of the original sources behind the Polishchuk-Spielman lemma lineage used in ArkLib's
coding-theory development.
In the repo it functions mainly as historical provenance for the decoding criterion formalized under
`Data/CodingTheory/PolishchukSpielman`.

- Historical source lineage for the Polishchuk-Spielman lemma formalized in
